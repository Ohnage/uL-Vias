**uL Vias** is a Blish HUD Pathing marker pack designed as a **living trail** for the **Underground Legends** community and anyone else who wants high‑signal, low‑friction paths for leveling and farming in Guild Wars 2.  

Version: **v1.1.8** – **Mar 29, 2026**

---

## What is uL Vias?
uL Vias is a curated set of TacO/Blish HUD markers focused on:
- **Speed leveling routes** tuned for clarity and repeatability  
- **High‑value resource farms** (especially bulk cooking supplies)  
- **Short, repeatable chest runs** for efficient looting

## Current contents
### uL Speed Leveling
**Category:** `uL Vias → Speed Leveling → Shiverpeak Mountains`

A structured leveling trail in Shiverpeak Mountains with:
- **Tutorial (WIP) – Queensdale**  
  - Optional tutorial markers in Queensdale (currently a work in progress).
- **Extra Steps [Map] overlay**  
  - Optional “extra steps” images on the edges of the world map to help you see the full arc of the route at a glance.
- **Damage‑type aware trading post hints**  
  - Toggle whether you’re leveling as **Power** or **Condition Damage**.  
  - This choice influences the **Trading Post messaging** baked into the route so recommendations line up with your build plan.
- **Volumes 1–5**  
  - The trail is split into **Volumes One–Five**, making it easy to run in chunks or repeat your favorite segments.

This part of the pack is ideal when you want a **repeatable, low‑decision‑load route** for fresh characters or alt leveling.

### Resource Farms
**Category:** `uL Vias → Resource Farms`

#### Bulk Cooking Supplies
A comprehensive **bulk cooking‑supplies atlas** with hand‑labeled NPCs and waypoints for quick restocking.

Each entry is a “X in Bulk” category with one or more vendors, clearly labeled by map, with copy‑ready waypoint instructions. Examples include:
- Almonds, Apples, Avocados  
- Bananas (multiple vendors and maps)  
- Basil Leaves, Bell Peppers, Black Beans (multiple locations)  
- Buttermilk, Celery, Cheese, Cherries  
- Chickpeas, Coconuts (core + Ember Bay), Cumin  
- Eggplants, Green Beans, Ginger Root  
- Horseradish Root, Kidney Beans, Lemons, Limes  
- Nutmeg Seeds, Peaches, Pears, Pinenuts  
- Rice, Shallots, Sour Cream, Tomatoes, Yeast  

Each vendor marker includes:

- **Map name**  
- Short instructions like “Talk to \<NPC\>, [direction] of \<Waypoint\>”  
- A **copy‑ready text** snippet you can paste into chat to waypoint yourself or your party.

This is meant to be your **one‑stop bulk cooking reference**: no alt‑tabbing to the wiki every time you need specific ingredients.

## Installation
You’ll need:

1. **Guild Wars 2**  
2. **Blish HUD** installed and running  
3. The **Pathing** module enabled (this is what reads TacO/Blish marker packs)

### 1. Install Blish HUD and Pathing
1. Download and install **Blish HUD** from the official site.  
2. In Blish HUD, open the **Module Repo**.  
3. Install and enable the **Pathing** module.

If you’re unsure, there are many community guides that walk through installing Blish HUD and enabling the Pathing module step‑by‑step.

### 2. Install the uL Vias marker pack
**Manual install (recommended to start):**

1. Download the latest `uL-Vias.taco` (or `.zip`) from the **Releases** page of this repository.  
2. Place the file into your Blish HUD markers folder, typically:

   - `C:\Users\<You>\Documents\Guild Wars 2\addons\blishhud\markers`  

   Community guides for other packs (like Tekkit’s All‑In‑One) use the same folder for manual marker install.

3. **Do not unpack** `.taco` files. Just drop them in the `markers` folder.
4. Restart Blish HUD (or reload markers) so Pathing can pick up the new pack.

## How to use in Blish HUD
1. Make sure **Pathing** is enabled.  
2. In Blish HUD’s Pathing UI, open the **markers list**.  
3. Look for **uL Vias** (or your configured pack name).  
4. Toggle categories on/off:
   - `uL Vias → Speed Leveling → Shiverpeak Mountains`  
   - `uL Vias → Resource Farms → Bulk Cooking Supplies / Flax`  
   - `uL Vias → Chest Farms → Icebound Chest`  

Use the in‑game toggles for:
- Turning the **Tutorial** on/off  
- Enabling **Extra Steps [Map]** overlays  
- Choosing **Power** vs **Condition Damage** leveling mode
